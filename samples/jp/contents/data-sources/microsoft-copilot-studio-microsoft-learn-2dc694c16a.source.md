# 概要 - Microsoft Copilot Studio | Microsoft Learn

Copilot Studio は、エージェントとエージェント フローを構築するためのグラフィカルでローコードなツールです。

[![Copilot Studio ホーム ページのスクリーンショット.](media/fundamentals-what-is-copilot-studio/home-page.png)](media/fundamentals-what-is-copilot-studio/home-page.png#lightbox)

Copilot Studio の目立つ機能の 1 つは、事前構築済みコネクタまたはカスタム コネクタを使用して他のデータ ソースに接続できることです。 この柔軟性により、洗練されたロジックを作成・調整でき、エージェント体験を強力かつ直感的に提供できます。

このプラットフォームのローコード体験は、AIの力を手元に置き、高度な技術的知識がなくてもアクセスしやすい環境を提供します。

## エージェントとは?

エージェントは、さまざまなやり取りやタスクを扱うことができる強力なAIコンパニオンです。 複雑な会話が必要な問題を解決し、指示や文脈に基づいて自律的に最善の行動を決定できます。 言語モデルのほか、指示、文脈、知識源、トピック、ツール、入力、トリガーを調整して目標達成をサポートします。

エージェントは、Web サイト、モバイル アプリ、Facebook、Microsoft Teams、またはAzure Bot Serviceでサポートされている任意のチャネルにまたがる複数の言語で顧客や従業員と連携できます。 さらに、会話の一環としてタスクを実行したり、トリガーに反応してユーザーや組織を支援したりして、生産性を向上させることもできます。

データ サイエンティストや開発者を必要とせずに、Copilot Studio でエージェントを簡単に作成できます。 エージェントを使用する方法の一部を次に示します。

- 営業支援とサポートの問題
- 営業時間と店舗情報
- 従業員の健康および休暇給付
- 公衆衛生の追跡情報
- 企業向けのよくある従業員質問

エージェントを単独で使用するか、エンタープライズ データとシナリオでMicrosoft 365 Copilotを拡張します。

## エージェント フローとは何ですか？

エージェント フローは、反復的なタスクを自動化し、アプリとサービスを統合するための強力な方法を提供します。 エージェント フローは、手動でトリガーすることも、他の自動化されたイベントやエージェントによってトリガーすることも、スケジュールに基づいてトリガーすることもできます。

Copilot Studio では、自然言語またはビジュアル エディターを使用してエージェント フローを作成できます。

エージェントフローはスタンドアロンの自動化として実行できます。 また、エージェントフローをツールとしてトリガーし、結果を同じエージェントに返すように設定することもできます。

## エージェントの会話はどのように機能しますか?

Copilot Studio エージェントは、カスタマイズされた NLU モデルと AI 機能を使用して、ユーザーが何を入力または言うかを理解し、最適なトピックで応答します。 トピックは、ユーザーとエージェント間の会話スレッドの一部です。 詳細情報は、[トピックを作成および編集する](authoring-create-edit-topics)を参照してください。

たとえば、顧客がビジネスに関する一般的な質問をするエージェントを作成することができます。 エージェントはサポート コールを転送することでサポートのオーバーヘッドを削減します。 エージェントでは、店舗の営業時間に関するトピックを作成し、**店舗の営業時間**という名前をそのトピックに付けることができます。

顧客が「いつ開店しますか？」や「営業時間は何時ですか？」などの質問をすると、エージェントは自然言語理解 (NLU) を使用して、質問の背後にある *意図* を理解します。 エージェントは、その意図を最適なトピックである **営業時間** トピックに一致させます。

エージェントは、*ストア時間*トピックで定義する**会話フロー** (接続されたノードのグループ) に従います。 一部のノードでは質問をしたり、条件 (if/else) を使用して顧客が希望するストアを決定したりすることがあります。 トピックの最終出力には、その特定の店舗の営業時間と連絡先情報が表示されます。

しかし、顧客がどんな質問をするかすべてを予測することはできません。 この問題を軽減するために、Copilot Studio には、NLU モデルの最新の進歩を使用する強力な AI を利用した機能が組み込まれています。 エージェントがナレッジソースにリンクされると、自動的に応答を生成できます。 これらの回答は会話形式で平易な言葉で、すべての事態に備えてトピックを作る必要はありません。

また、エージェントが知識源外の情報にアクセスできるようにすることも可能です。

Copilot Studio では、Bingでも使用される Azure OpenAI GPT モデルを利用した AI を使用して、ニーズの簡単な説明からトピックを作成できます。 同様に、変更したい内容を記述することで、エージェント内の任意のトピックを変更および更新できます。

## Access Copilot Studio

スタンドアロン Web アプリとして、または Teams 内の個別のアプリとして、Copilot Studio にアクセスします。 Teams 用の Copilot Studio アプリは、従来のチャットボットのみをサポートしています。

### Web アプリ

利用事例

- あなたはIT管理者で、タスクを実行したり顧客とやり取りしたりするためのエージェントを作成したいと考えています。
- エージェント サービスに慣れているので、Copilot Studio を試用またはテストする必要があります。
- エンティティや変数などの高度なエージェントの概念を探求し、複雑なエージェントを作成したいのです。

ウェブアプリにアクセスしてください。 https://copilotstudio.microsoft.com

[Copilot Studio のデモを体験する](https://copilotstudio.microsoft.com/tryit?azure-portal=true)

### Teams アプリ

利用事例

- あなたは組織の従業員またはメンバーで、チャットボットを使ってよくある従業員の質問に答えたいと思っています。
- エンティティや変数などの高度な概念を使い、Teams内でチャットボットを内部的に利用できるようにしたいです。
- チャットボットを素早く作成し配布したいのです。

[Teams で Copilot Studio アプリを開くか追加します](https://aka.ms/PVATeamsApp?azure-portal=true)

## エージェントを計画する

エージェントを計画する際には、以下の点を考慮してください。

### エージェントでMicrosoft 365 Copilotを拡張する

次の場合は、エージェントでMicrosoft 365 Copilotを拡張することを検討してください。

- 手順、ツール、知識を宣言して独自のエージェントを作成し、特定のタスクとドメインの知識に合わせてMicrosoft 365 Copilotをカスタマイズする必要があります。
- 既存のCopilot オーケストレーターを利用する必要があります。
- スタンドアロンのカスタム バージョンのMicrosoft 365 Copilot チャット エクスペリエンスが必要です。

### エージェントを作成する

Copilot Studio を使用すると、エージェントを簡単に作成できます。 欲しいエージェントを分かりやすく説明すれば十分です。 Copilot Studio に、エージェントに必要な具体的な指示、トリガー、ナレッジ ソース、ツールを伝えます。 その後、エージェントを展開する前にテストしてください。 準備ができたら、複数のチャネルでエージェントを公開します。

次の場合にはエージェントの作成を検討してください。

- 次のことが可能なエージェントが必要です。

    - 会社のデータとドキュメントを統合する
    - 外部 API からリアルタイム データを取得する
    - 外部イベントに応答するアクションの実行
    - 会社のアプリケーションに埋め込む
- 特定のビジネス ニーズを満たし、製品のブランド化を完全に制御できる、Web アプリやモバイル アプリ、自動化ワークフロー向けのカスタマイズされたエンドツーエンドのソリューションが必要です。
- あなたのエージェントを、他のエージェントのサポート対象となるエージェント拡張機能として公開したい場合。
- あなたは熟練した開発者であり、ビジネス ニーズに応えるためにカスタマイズされたエンド ツー エンドのソリューションを作成しようとしており、次のことを望んでいます。

    - 製品ブランドの完全なコントロール
    - 言語モデルとオーケストレーションの選択

    あるいは、次のような製品を構築している場合:

    - 電子商取引サイト向けのカスタマーサービスチャットボット
    - 医療サービスの予約スケジュールをする仮想アシスタント
    - 生成 AI を取り入れたゲーム体験
