# mdi — Business Icons

Curated subset of **200** business-relevant icons from the `mdi` Iconify collection.

Use these as `icon` values in slide definitions. Example: `mdi:access-point-check`.

Cached SVGs are stored in `cache/mdi/` for offline PPTX generation.
